How It Works

1

Client-Side Encryption

Your file is encrypted in your browser using AES-256-GCM before it ever leaves your device. The encryption key is generated locally and never sent to our servers. Powered by the Web Crypto API built into your browser.

2

Pay with Lightning

Pay a small fee (1 sat/MB, minimum 100 sats) via the Lightning Network to host your encrypted file for 7 days.

3

Share Securely

The decryption key is stored in the URL fragment (after the #), which is never sent to the server. Only people with your exact link can decrypt the file.
